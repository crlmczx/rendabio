# RendaBio 完整部署指南

## 🎯 目标
部署网站到 **www.rendabio.com**

## 方案一：一键自动部署（推荐）

### 准备工作

1. **获取 GitHub Token**
   - 访问: https://github.com/settings/tokens
   - 点击 "Generate new token (classic)"
   - 勾选 `repo` 权限
   - 生成并复制 Token

2. **获取 Vercel Token**
   - 访问: https://vercel.com/account/tokens
   - 点击 "Create Token"
   - 复制 Token

### 执行部署

```bash
cd /mnt/okcomputer/output/app
bash auto-deploy.sh
```

按提示输入：
- GitHub 用户名
- GitHub Token
- Vercel Token

脚本将自动完成：
- ✅ 创建 GitHub 仓库
- ✅ 推送代码
- ✅ Vercel 部署
- ✅ 绑定域名

## 方案二：手动部署（更可控）

### 步骤 1：创建 GitHub 仓库

访问 https://github.com/new

填写信息：
- Repository name: `rendabio`
- Description: `AI多智能体协作平台`
- Visibility: Public
- ✅ 勾选 "Add a README file"

点击 **Create repository**

### 步骤 2：推送代码

```bash
cd /mnt/okcomputer/output/app

# 添加远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/rendabio.git

# 推送代码
git push -u origin main
```

### 步骤 3：Vercel 部署

1. 访问 https://vercel.com
2. 点击 **Sign Up** → **Continue with GitHub**
3. 点击 **Add New Project**
4. 导入 `rendabio` 仓库
5. 点击 **Deploy**

### 步骤 4：绑定域名

1. 进入 Vercel 项目 → **Settings** → **Domains**
2. 输入 `www.rendabio.com` → **Add**
3. 在域名注册商处添加 DNS：

| 类型 | 主机记录 | 记录值 |
|------|---------|--------|
| CNAME | www | `cname.vercel-dns.com` |
| A | @ | `76.76.21.21` |

## 方案三：Vercel CLI 部署

```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login

# 进入项目目录
cd /mnt/okcomputer/output/app

# 部署
vercel --prod

# 绑定域名
vercel domains add www.rendabio.com
```

## 🆘 常见问题

### Q: 推送失败 "Permission denied"
**解决**: 使用 Token 方式推送
```bash
git remote set-url origin https://USERNAME:TOKEN@github.com/USERNAME/rendabio.git
git push
```

### Q: Vercel 部署失败
**解决**: 检查构建设置
- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`

### Q: 域名无法访问
**解决**: 
1. 检查 DNS 记录是否正确
2. 等待 DNS 传播（5-30 分钟）
3. 使用 `nslookup www.rendabio.com` 检查
